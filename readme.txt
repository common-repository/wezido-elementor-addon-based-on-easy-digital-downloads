 An Elementor Extension Works with the Easy Digital Download Products

== Description ==

Add more power to your Elementor page builder experience with EDD by using our easy to use widgets.
Consolidate these creative elements in your WordPress website to make it look more beautiful and friendly than before.

== Wezido - Elementor Addon Features ==

We have built 5+ useful widgets to enhance your Elementor Page Building experience. More coming soon!

1. [Headings](https://plugins.teconcetheme.com/wezido/index.php/headings/) - You will be able to choose any kind of heading tags.
also, there are many styles with a bunch of customization controls.

2. [Team](https://plugins.teconcetheme.com/wezido/index.php/team/) - Wezido has team elements with customizable huge options.

3. [Flip Box](https://plugins.teconcetheme.com/wezido/index.php/flipbox/) - Flipbox available with bunch of customization control.
also flipbox has various animation options too.

4. [Before After](https://plugins.teconcetheme.com/wezido/index.php/before-after/) -There Available before after slider.
so you can easily show the slides.

5. [Blog Post](https://plugins.teconcetheme.com/wezido/index.php/blog-post/) - You will be able to show your blog post based on category, tags etc.

6. [Blog Category](https://plugins.teconcetheme.com/wezido/index.php/blog-catgegory/) - You will be able to show your blog category with featured image.

7. [Counter](https://plugins.teconcetheme.com/wezido/index.php/counter/) - Amazing counter available with Edd sales, product & site user count.

8. [Pricing Table](https://plugins.teconcetheme.com/wezido/index.php/pricing-table/) - Pricing table available with lots of customization options.

9. [EDD Product](https://plugins.teconcetheme.com/wezido/index.php/edd-product/) - You will be able to display your EDD digital products to any pages.

10. [EDD Justified](https://plugins.teconcetheme.com/wezido/index.php/edd-justified-grid/) - You will be able to display your EDD digital products in Justified grid mode on any page.

11. [EDD Masonry](https://plugins.teconcetheme.com/wezido/index.php/edd-masonry/) - You will be able to display your EDD digital products in Masonry grid mode on any page.

12. [EDD Category](https://plugins.teconcetheme.com/wezido/index.php/edd-category/) - You will be able to display your EDD Category to any pages.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
